# The Comfort Slider

A playful iOS app that answers one question: **what salary would you need to
comfortably — or very *un*comfortably — afford this?**

Enter a total price and your financing (down payment, APR, term, and optional
sales tax), then drag the comfort slider from **Comfortable → Stretching →
Unhinged**. The app works out the monthly payment and the annual salary that
purchase implies at your chosen comfort level (via debt-to-income), complete with
money-spray animations, haptics, and a roast for your decision.

Set a default salary in Settings and it flips around: the slider snaps to show how
(un)comfortable a purchase is *at your salary*, and moves as the price changes —
you can still drag it to explore.

Built with SwiftUI, Liquid Glass surfaces, and an animated Metal shader background.

## Getting started

**Prerequisite:** [Tuist](https://tuist.dev). Install it via the
[official guide](https://docs.tuist.dev/en/guides/quick-start/install-tuist):

```sh
curl -Ls https://install.tuist.io | bash
```

The Xcode project is generated by Tuist and is not committed. From the
`the-comfort-slider/` directory, run:

```sh
tuist install    # resolves Swift Package dependencies
tuist generate   # generates and opens the Xcode project
```

Then build and run the `the-comfort-slider` scheme. Haptics, glass, and the shader
require a real iOS 26 simulator or device (they don't render in previews).
